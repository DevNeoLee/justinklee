Share Desk

Workspace rental platform that enables users to list and book private workspaces, offices, and desk spaces for short-term rental. The application features advanced location-based search with Google Maps integration, real-time availability tracking, secure payment-ready reservation system, and social authentication through Google and Facebook OAuth. Users can upload multiple workspace images, manage bookings through an intuitive dashboard, and leave star-rated reviews. The platform includes sophisticated search filters, infinite scroll pagination, and responsive mobile-first design to ensure seamless experience across all devices. Full-stack implementation utilizes Ruby on Rails 7.1.5.1 with Ruby 3.3.0, PostgreSQL for data persistence, and Stimulus.js for modern JavaScript interactions. The frontend leverages Bootstrap 5 for responsive UI components, while backend services integrate with Google Maps API for geolocation services, AWS S3 for scalable file storage, and Render.com for cloud deployment. Technological significance includes implementing fallback systems for critical APIs to ensure production reliability, developing custom Stimulus controllers for infinite scroll and real-time form handling, and creating a robust authentication system with Devise and OmniAuth. The application demonstrates modern Rails practices with importmaps for JavaScript module management, Active Storage for file handling, and Ransack for advanced search capabilities. The architecture showcases scalable cloud deployment strategies and comprehensive error handling for enterprise-level reliability.

Collectors

Full-stack e-commerce application for premium movie and media collectors, showcasing mastery of modern React development patterns and enterprise-level architecture. The application features a comprehensive product catalog spanning five specialized categories (Movies, Animation, TV Series, Rare Collections, Sports/Arts) with over 40 premium products, implementing advanced state management through Redux Toolkit with persistent cart functionality and real-time user authentication via Firebase Auth. Built with React 19 and TypeScript for type safety, the application demonstrates advanced performance optimization techniques including React.memo for component memoization, lazy loading with React.Suspense for code splitting, and custom hooks for logic reusability. The architecture implements Redux Toolkit's modern state management patterns with selective persistence using redux-persist, while integrating Stripe Checkout for secure payment processing. The application showcases responsive design principles with SCSS Modules for component-scoped styling, comprehensive error handling through Error Boundaries, and accessibility features including ARIA labels and keyboard navigation. The Firebase integration demonstrates real-time data synchronization with Firestore, implementing efficient user profile management and authentication state persistence. The development workflow utilizes Vite for rapid development and optimized builds, while maintaining clean code architecture through ESLint configuration and TypeScript strict mode compliance.

Travelers to Canada

Fully interactive data visualization dashboard built with TypeScript, D3.js (v7), and Vite, designed to present 20 years of Canadian tourism statistics in a single, responsive web page. Leveraging modern web technologies and best practices—including modular TypeScript architecture, CSS Grid/Flexbox layouts, and scalable SVG graphics—the application delivers a seamless user experience where all key insights (bar chart, pie chart, map, and controls) are visible at a glance. Key technical highlights include advanced D3.js charting (threshold, linear, and square-root scales), real-time data filtering, and a robust, maintainable codebase with strict linting and type safety.
