Share Desk

Share Desk App is a global platform that allows users to instantly find and book workspaces anywhere in the world by searching for local availability. Users can reserve desks in real time, while hosts have the opportunity to earn income by listing unused spaces in their homes, apartments, or offices. The application is built with a Ruby on Rails backend and utilizes Turbolinks and Stimulus.js for a fast and dynamic front-end experience. Infinite scrolling is implemented through Ajax calls, and JavaScript and jQuery are used for DOM manipulation and interactivity. The design is responsive, using both Flexbox and CSS Grid to ensure a smooth user experience across devices. Data is stored in a PostgreSQL database and deployed via Heroku. User authentication is handled through Google and Facebook using OmniAuth. The app integrates the Google Maps API with Geocoding to help users locate spaces easily. File storage is managed through Amazon AWS S3. Key features include a reservation system with a datepicker, custom user profiles, the ability to host and manage listings, advanced search functionality, and a customer star rating system. The platform draws inspiration from industry leaders like WeWork and Airbnb, aiming to provide a seamless and modern experience for both guests and hosts.
